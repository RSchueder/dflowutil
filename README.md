# dflowutil
Utilities for Delft3D FLOW Flexible Mesh in python. It contains 
utilities useful for hydrodynamic and water quality simulations

A list of implemented functions is as follows:
 *boundary_from_ext()
 *change_os()
 *dflow_grid_2_tri()
 *find_last()
 *find_limit_cell()
 *nc_format()
 *nc_station()
 *pdistf()
 *plot_nc_map()
 *read_bc()
 *read_pli()
 *read_polygon()
 *read_sub_file()
 *row2array()
 *show_waq_segment()
 *write_lsp_table()
